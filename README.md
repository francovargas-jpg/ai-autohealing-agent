# Infrastructure Auto-Healing Agent

Multi-agent AI system that detects failures in AWS infrastructure, diagnoses the root cause, and acts autonomously — without human intervention.

## Stack
- Amazon EKS
- Amazon Bedrock
- AWS Lambda
- AWS Step Functions
- Amazon EventBridge
- Amazon CloudWatch
- Amazon ECR
- Terraform
- Jenkins

## Project Structure
- `infra/` — Terraform infrastructure as code
- `functions/` — Lambda functions (agents)
- `docs/` — Architecture diagrams and documentation
